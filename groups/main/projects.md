# Project Registry

This file maps project names to their GitHub repositories. Use these short names when referencing projects.

## Projects

| Name | Repository | Description |
|------|------------|-------------|
| verychess | https://github.com/dbm87tech/verychess | Chess application |

## How to Use

When a user mentions a project by name (e.g., "the webapp project", "verychess"), look it up here to get the full repository URL.

When adding new projects, update this file with:
- **Name**: Short identifier (lowercase, no spaces)
- **Repository**: Full GitHub URL
- **Description**: Brief description

## Adding a Project

To add a new project, add a row to the table above:
```
| myproject | https://github.com/org/myproject | Description here |
```
